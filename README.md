# useful-shell-script

有用的shell脚本

## backup_sql_via_git

此脚本可以通过git来备份你的数据库，并附带注释（如果需要）

```shell
./backup [comment]
```

## backup_sql_to_extend_storage

该脚本可以在备份后移动备份内容至你想要的地方
